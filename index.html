6502 Assembler (NESASM3)

Numbers

  • $00 - Zero page address
  • $0200 - 16 bit address
  • #$0F - Hexidecimal literal
  • #%10001010 - Binary literal

Registers

  • A - Accumulator Register
  • X - Index Register X
  • Y - Index Register Y
  • Status Register

Opcodes and Labels

Directives and Labels

Loading and Storing

  • LDA #$0F
  • LDX $01
  • LDY $4016
  • STA $2000
  • STX $00
  • STY $00

Arithmetic

  • ADC #$01 | A = A + $01 + carry
  • CLC | Clear carry
  • SBC #$08 | A = A - $08 - (1 - carry)
  • SEC | Set carry
  • INC $0100 | Increment value stored by 1
  • DEC $0001 | Decrement value stored by 1
  • INX | X = X + 1
  • DEX | X = X - 1
  • ASL A | Shift left 1 bit, quick x2
  • LSR A | Shift right 1 bit, quick /2

Comparison

  • CMP #$00 | Compare A to the value $01
  • CPX $01 | Compare X to the value stored in addr $01
  • CPY #%10101010 | Compare Y to the binary literal

Flow Control

  • JMP $8500
  • BEQ $FF00
  • BNE $FF00
  • JSR MyFunctionLabel
  • RTS
